# ansi cheatsheet

provides a list of `\x1b[...m` where `...` is all integers from 0 to 127, as to provide a cheatsheet for ansi usage.

generated from [cheatsheet.js](./cheatsheet.js).

## usage

`curl ansi.paws.ong` or `wget -qO- ansi.paws.ong`
